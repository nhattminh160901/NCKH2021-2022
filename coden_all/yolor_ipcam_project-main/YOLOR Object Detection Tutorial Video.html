Step 1 - Install Prerequisites
Ensure that you have the following for both GPU and CPU Installations:

PyCharm Community
Anaconda
Ensure that you have the following installed if you are doing the GPU Implementation:

NVIDIA Drivers (Updated to Latest)
CUDA Toolkit 11.1

Step 2.1 - GPU Installation
Create Conda Environment 
conda create --name yolor-gpu

conda activate yolor-gpu

Install PyTorch, Cython and PyCocoTools 
pip install torch==1.9.0+cu111 torchvision==0.10.0+cu111 torchaudio===0.9.0 -f https://download.pytorch.org/whl/torch_stable.html

pip install -U cython

pip install "git+https://github.com/philferriere/cocoapi.git#egg=pycocotools&subdirectory=PythonAPI"

Change Directory to Downloaded Files 
pip install -r requirements-gpu.txt

Step 2.2 - CPU Installation
Create Conda Environment 
conda create --name yolor-cpu

conda activate yolor-cpu

Install Dependencies 
pip install seaborn thop

pip install -r requirements-cpu.txt

pip install "git+https://github.com/philferriere/cocoapi.git#egg=pycocotools&subdirectory=PythonAPI"

Step 3 - Run on Images
GPU 
python detect.py --source inference/images/horses.jpg --cfg cfg/yolor_p6.cfg --weights yolor_p6.pt --conf 0.25 --img-size 1280 --device 0

CPU
python detect.py --source inference/images/horses.jpg --cfg cfg/yolor_p6.cfg --weights yolor_p6.pt --conf 0.25 --img-size 1280 --device cpu

Step 4 - Run on Webcam
Ensure that Webcam is connected.

GPU
python detect.py --source 0 --cfg cfg/yolor_p6.cfg --weights yolor_p6.pt --conf 0.25 --img-size 1280 --device 0

CPU
python detect.py --source 0 --cfg cfg/yolo

@Benya, it seems that the installed version of opencv you have is missing some modules. Secondly, did you build opencv in ubuntu or just install it.

Way-1:
If you built opencv in ubuntu from scratch with Cmake, then you will first need to install libqt4 package with mentioned command.
sudo apt-get install libqt4-dev
and then you will need to check mentioned option in Cmake-gui. 
WITH_QT=ON
After this make sure to build opencv again.
Way-2:
If you just installed opencv, then uninstall opencv, and reinstall it with third parties modules. you can run mentioned commands to accomplish this step.
pip uninstall opencv-python
or
sudo apt-get uninstall opencv-python

sudo apt-get autoremove
sudo apt-get clean
sudo apt-get install libopencv-*
pip install opencv-python
or
sudo apt-get install opencv-python
For more details, about this issue you can visit mentioned link.
